The Gym Management System is a full-stack web application built with a modular architecture. 
It is designed to streamline fitness center operations by providing distinct, dedicated portals for both administrators and regular users, allowing for efficient management of facility operations and personalized fitness journeys.
Technologies Used:
Database: MongoDB
Backend: Node.js and Express.js
Frontend: React
Stack: MERN

How To Run It?

Step1:
First of all run 'npm install' to install all the required dependencies for backend and frontend.

Step2:
Open the folder in the code folder.

Step3:
Make sure that MongoDB is installed and running on your system.

Key Features:-

Role-Based Portals: Separate, dedicated interfaces for Administrators (to manage the facility and members) and Users (to access their personal plans).
Comprehensive Data Management: Custom database schemas to effectively track and manage user profiles, workout plans, diet schedules, and attendance.
Authentication & Security: Role-based access control and authentication to ensure data privacy and secure logins for different user types.
Financial Processing: Integrated mock payment processing to simulate membership billing and transactions.
Interactive Dashboards: Robust API routing connecting the backend to the frontend to power dynamic dashboard interactions for both admins and members.
